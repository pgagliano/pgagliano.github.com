Appointment Manager (C++)

An appointment manager that allows the user to perform a set of tasks. This application uses a myDate class and a myTime classes. The application creates an array of 50 myDate objects. When the application begins, it will prompt the user to read in from the date.dat file and partially fills the array. The program will then display a menu for the user to select from a list of tasks.
